# Arduino Projects

This repository contains Arduino and ESP32 projects that I made while learning embedded systems and electronics.
So far, I have worked with sensors, motors, LEDs, Bluetooth communication, Wi-Fi connectivity, and basic robotics. Some projects include controlling LEDs and motors using Bluetooth, IR sensors, and web interfaces, as well as using ultrasonic sensors for distance measurement and obstacle detection.

The purpose of this repository is to keep track of my learning, experiments, and mini projects as I continue exploring Arduino programming and IoT development.
Feel free to check out the code and try the projects yourself.
